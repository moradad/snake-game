# 🐍 贪吃蛇大作战 Snake Game

> 一个纯前端实现的贪吃蛇小游戏，支持闯关模式 & 自由模式，带本地/在线排行榜、按键自定义、音量控制等。

---

## 🚀 快速开始

1. **Clone**
   ```bash
   git clone https://github.com/moradad/snake-game.git
   cd snake-game

---
## 🎮 功能特性
闯关模式：吃满 15 个食物过一关，关卡升级时速度提高、刷新障碍

自由模式：无限吃食物，随时结束

本地 & 在线排行榜：支持日/周/月/总榜

按键自定义：可修改上、下、左、右、加速、暂停键

音量 & 皮肤：背景乐与音效音量独立调整，多种蛇皮肤可选
